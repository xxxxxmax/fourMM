# ALMM — Autonomous Liquidity Market Maker

> Agent-first CLI for Four.meme market making on BSC.

**Status:** 🚧 WIP — Week 1 scaffolding in progress.

## What it does

ALMM is a command-line tool for running market-making strategies on [Four.meme](https://four.meme) tokens on BNB Smart Chain, designed to be driven by AI agents (Claude Code / OpenClaw / MCP clients).

### The four things that make ALMM different

1. **OWS policy-gated keys** — Treasury wallet is held by the [Open Wallet Standard](https://openwallet.sh); agents sign via scoped API tokens that are locked to BSC and auto-expire.
2. **Atomic Router contract** — A 120-line Solidity router on BSC does `createToken + dev buy` and `buy + sell round-trip` in a single tx, replacing Solana's Jito-Bundle atomicity that BSC lacks.
3. **Bonding curve ↔ PancakeSwap auto-routing** — Every trade command detects whether the token has graduated and routes through the correct contract. Your volume bot doesn't break at graduation.
4. **Refuses TaxToken / X Mode** — Because the math doesn't work. ALMM knows what real market making means.

## Quick start (coming soon)

```bash
# Initialize config
almm config init --rpc-url "https://bsc-dataseed.binance.org/"

# Create OWS-managed treasury
almm ows init --name treasury
almm ows policy setup --chain bsc --expires 24h
almm ows key create --name my-agent

# Create in-house wallet group
almm wallet create-group --name snipers --count 10

# Query a real Four.meme token
almm token info --ca 0x...4444
```
